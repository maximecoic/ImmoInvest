# ImmoSimulateur

ImmoSimulateur is a simple and intuitive real estate investment simulator for French citizens. It allows users to compare various investment strategies without the complexity of extensive financial jargon.

## Features
- Investment input fields (purchase price, rental income, tax bracket).
- Comparison output of investment options.
- Data stored locally via JSON files.
- Responsive design for mobile accessibility.

## File Structure
- `index.html`: Main entry point of the application.
- `styles.css`: Contains all styles for responsive design.
- `script.js`: JavaScript functions for user interactions.
- `data/`: Contains JSON files for investment options.

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in any web browser to start.
3. Use the simulator to analyze different investment options.
