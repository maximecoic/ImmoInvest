# InvestiSim

InvestiSim is a real estate investment simulator designed for French citizens to compare investment strategies such as LMNP, Denormandie, and Déficit Foncier. It features a user-friendly interface with input forms, data comparisons, and visualizations.

### Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env.local` file and set your Supabase URL and anon key.
4. Run `npm run dev` to start the development server and visit `http://localhost:3000`.

### Features

- User input fields for financial data
- Investment comparison engine
- Visual data representation using charts
- Scenario simulation capability

### Technology Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase for backend
- Axios for API requests
